  • Responsive — looks good on phones and desktops.

DEE to GitHub Pages

  1. Create a new repo on GitHub (or use username.github.io).
  2. Save this file as index.html in the repo root (or /docs/index.html).
  3. GIT Commit & push to the main branch.
  4. On GitHub, go to Settings → Pages and choose main (or /docs) as source.
  5. Visit https://username.github.io/repo (or https://username.github.io for a user site).
